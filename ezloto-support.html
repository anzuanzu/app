選號結果能保證中獎嗎？

不能。策略、日期、生日、姓名、星座、紫微斗數與包牌功能皆僅供娛樂、資料整理及個人參考，並不保證中獎。

分享 QR Code 可以用來投注嗎？

不可以。QR Code 僅用於個人分享與記錄選取的號碼，不能作為任何投注或通路掃描使用。

開獎資料何時更新？

App 啟動或手動重新整理時會檢查可用的開獎資料。資料顯示時間可能因來源更新或網路狀態而略有差異。

為什麼會看到廣告或隱私選項？

EZ大樂透以廣告支持免費提供。依所在地規則，App 可能顯示廣告隱私同意訊息，並在需要時提供日後修改隱私選項的入口。

如何儲存 QR Code 圖片？

在分享 QR Code 畫面選擇「儲存至相簿」。iPhone 會在第一次儲存時詢問新增照片權限；只有你主動操作時才會儲存圖片。

未滿 18 歲請勿購買彩券。若你對資料或隱私權有疑問，請閱讀隱私權政策或直接聯絡我們。
