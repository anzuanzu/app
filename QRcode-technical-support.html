應用程式功能

  • 即時掃描：使用相機即時掃描 QR Code
  • 拍照掃描：拍照後掃描 QR Code，適合光線不佳的環境
  • 圖片掃描：從相簿選擇圖片進行 QR Code 掃描
  • 分享掃描：從其他應用程式分享圖片到本 App 進行掃描
  • 結果處理：自動識別網址並提供開啟選項，支援複製到剪貼板
  • 震動回饋：掃描成功時提供震動提示

常見問題

❓ 為什麼無法掃描 QR Code？

可能原因及解決方法：

  • 確保 QR Code 清晰完整且在畫面中央
  • 檢查光線是否充足，可使用手電筒功能
  • 嘗試調整手機與 QR Code 的距離
  • 如果即時掃描有問題，可使用「拍照掃描」功能
  • 確保 QR Code 沒有損壞或變形

❓ 相機無法啟動怎麼辦？

解決步驟：

  • 檢查是否已授予相機權限：設定 → 隱私權與安全性 → 相機 → QR Code 掃描器
  • 重新啟動應用程式
  • 重新啟動手機
  • 確保沒有其他應用程式正在使用相機

❓ 如何使用分享功能掃描圖片？

使用方法：

  • 在其他 App（如照片、訊息、Safari）中找到包含 QR Code 的圖片
  • 點擊「分享」按鈕
  • 選擇「QR Code 掃描器」
  • App 會自動掃描並顯示結果

❓ 掃描結果無法開啟連結？

可能原因：

  • 確保網路連線正常
  • 檢查 QR Code 內容是否為有效的網址格式
  • 某些特殊格式的連結可能需要特定應用程式才能開啟
  • 可以選擇「複製」功能，手動貼到瀏覽器中開啟

❓ 應用程式閃退或運行緩慢？

故障排除：

  • 確保 iOS 系統版本為最新
  • 重新啟動應用程式
  • 重新啟動手機
  • 檢查手機儲存空間是否充足
  • 如果問題持續，請卸載後重新安裝應用程式

系統需求

  • iOS 12.0 或更新版本
  • 具備相機功能的 iPhone 或 iPad
  • 網路連線（用於開啟掃描到的網址）

隱私權與安全

  • 我們不會儲存您掃描的 QR Code 內容
  • 所有掃描處理均在您的裝置上進行
  • 相機權限僅用於 QR Code 掃描功能
  • 詳細資訊請參考我們的隱私權政策
