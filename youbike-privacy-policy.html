隱私權政策  ｜  Privacy Policy

最後更新：2026 年 4 月 15 日  ｜  開發者：J-Studio

概述

J-Studio 開發的「YouBike 即時地圖」（以下稱「本 App」）非常重視您的隱私。本政策說明本 App 如何收集、使用及保護您的個人資訊。

使用本 App 即表示您同意本政策的內容。若您不同意，請停止使用本 App。

收集的資訊

位置資訊

  • 本 App 在使用期間會取得您的裝置位置（使用中授權）。
  • 位置資訊僅用於顯示附近 YouBike 站點、計算距離及更新 Widget 最近站點。
  • 位置資訊不會傳送至本 App 的伺服器，亦不會與第三方共享（廣告服務除外，詳見下方）。

廣告識別碼（IDFA）

  • 本 App 整合 Google AdMob 廣告服務。
  • 若您同意追蹤，AdMob 可能使用廣告識別碼（IDFA）提供個人化廣告。
  • 若您拒絕追蹤，AdMob 仍可能投放非個人化廣告，但不會使用 IDFA。
  • 您可隨時在「設定 > 隱私權與安全性 > 追蹤」變更此設定。

使用資料

  • 本 App 不會主動收集、儲存或傳送任何個人身份資訊。
  • 收藏站點資料僅儲存於您的裝置本機（App Group 儲存空間）。

資訊的使用方式

  • 提供 YouBike 站點即時資訊與距離計算
  • 顯示 Widget 最近站點資訊
  • 透過 Google AdMob 投放廣告（維持 App 免費營運）

本 App 不會出售、租借或以任何形式交易您的個人資訊。

第三方服務

交通部 TDX 運輸資料流通服務

  • 本 App 透過 Cloudflare Worker 代理向 TDX API 取得 YouBike 站點資料。
  • 請求過程不包含任何個人資訊。

Google AdMob

  • 本 App 使用 Google AdMob 提供廣告服務。
  • AdMob 可能依據您的裝置資訊及使用行為投放廣告。
  • 詳細說明請參閱 Google 隱私權政策。

Cloudflare

  • API 請求透過 Cloudflare Worker 代理，Cloudflare 可能記錄標準存取日誌（IP、時間戳記等）。
  • 詳細說明請參閱 Cloudflare 隱私權政策。

資料安全

  • API 憑證（TDX Client ID / Secret）儲存於伺服器端，不存於裝置上。
  • 所有 API 通訊均透過 HTTPS 加密傳輸。
  • 收藏站點等本機資料儲存於 iOS App Group，受系統沙盒保護。

兒童隱私

本 App 適用於一般大眾，年齡分級為 4+。本 App 不會故意收集 13 歲以下兒童的個人資訊。

您的權利

  • 拒絕位置存取：可在「設定 > 隱私權與安全性 > 定位服務」關閉，關閉後地圖將預設顯示台北車站附近。
  • 拒絕廣告追蹤：可在「設定 > 隱私權與安全性 > 追蹤」關閉。
  • 刪除本機資料：解除安裝 App 即可移除所有本機儲存資料。

政策變更

本隱私權政策可能因應法規或功能更新而修訂。修訂後將更新本頁面的「最後更新」日期。建議您定期查閱。
